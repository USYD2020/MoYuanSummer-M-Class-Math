🔢

和差倍问题

已知两个数的和、差、倍三个量中的两个，求这两个数的问题

➕

和差问题

已知两数的和与差，求这两个数的问题

✖️

和倍问题

已知两个数的和以及它们之间的倍数关系，求这两个数的问题

➖

差倍问题

已知两个数的差，它们之间的倍数关系，求这两个数的问题

📏

线段图法

找出一倍量，用线段表示，标记"△"，算总份数及对应总和

🎯

份数思想

把较小量看作1份，较大量看作几份，转化为求"总份数"

🌟

解题口诀

有倍先画倍、再找和与差；和、差的时间状态要与倍保持一致

技巧总结

六种核心技巧，掌握和差倍问题计算精髓

🍓 基本解法公式

➕ 和差问题公式

较小数 = (和-差)÷2
较大数 = (和+差)÷2

✖️ 和倍问题公式

较小数 = 和÷(倍数+1)
较大数 = 较小数×倍数

➖ 差倍问题公式

较小数 = 差÷(倍数-1)
较大数 = 较小数×倍数

🥝 高级应用技巧

🔄 等量代换法 / 解方程

用倍数关系表示未知量，构造整倍

份数思想 ：
较小量看作1份，较大量看作几份，转化为求"总份数"

🎯 转化法

通过添加、减少、调整"和"或"差"，转化为标准问题

暗和：给来给去和不变
暗差：同增同减差不变

【范例4：周长问题】

一个蚂蚁沿着一个长方形的花盆爬圈，爬了1圈，一共爬了240厘米，已知花盆长和宽相差10厘米，请问这个花盆的长和宽各是多少？

【范例4：周长问题】

一个蚂蚁沿着一个长方形的花盆爬圈，爬了1圈，一共爬了240厘米，已知花盆长和宽相差10厘米，请问这个花盆的长和宽各是多少？

🎯 转化思路

周长240厘米，长+宽=240÷2=120厘米，转化为和差问题

✅ 解析

和：240÷2 = 120，差：10

宽：(120-10)÷2 = 55厘米

长：55+10 = 65厘米

【范例5：变化后的和倍】

公园里有牡丹花和月季花共22朵，又种3朵牡丹花以后，牡丹花是月季花的4倍，那么原来两种花各有多少朵？

【范例5：变化后的和倍】

公园里有牡丹花和月季花共22朵，又种3朵牡丹花以后，牡丹花是月季花的4倍，那么原来两种花各有多少朵？

🔄 转化提示

变化后总数：22+3=25朵，然后用和倍问题解决

✅ 解析

和：22+3 = 25，倍数：4

月季花：25÷(4+1) = 5朵

牡丹花：4×5-3 = 17朵

【范例6：变化后的差倍】

小洪和小陈在下围棋，棋盘上小洪的棋子数量是小陈的4倍。如果拿掉小洪24颗棋子，两人的棋子一样多。小洪和小陈原来各下了多少颗棋子？

【范例6：变化后的差倍】

小洪和小陈在下围棋，棋盘上小洪的棋子数量是小陈的4倍。如果拿掉小洪24颗棋子，两人的棋子一样多。小洪和小陈原来各下了多少颗棋子？

🎯 逆向思维

拿掉24颗后相等，说明原来小洪比小陈多24颗

✅ 解析

差：24，倍数：4

小陈：24÷(4-1) = 8颗

小洪：24+8 = 32颗

【范例7：倍数关系问题】

水果店运进苹果和香蕉共48千克。其中香蕉比苹果多了4倍，问：运进来苹果和香蕉各多少千克？

【范例7：倍数关系问题】

水果店运进苹果和香蕉共48千克。其中香蕉比苹果多了4倍，问：运进来苹果和香蕉各多少千克？

⚠️ 注意理解

"多了4倍"意味着香蕉是苹果的4+1=5倍

✅ 解析

和：48，倍数：4+1 = 5

苹果：48÷(5+1) = 8千克

香蕉：8×5 = 40千克

【范例8：转移数量问题】

两筐梨子共有120个，如果从第一筐中拿10个放到第二筐中，那么两筐的梨子个数相等。两筐原来各有多少个梨？

【范例8：转移数量问题】

两筐梨子共有120个，如果从第一筐中拿10个放到第二筐中，那么两筐的梨子个数相等。两筐原来各有多少个梨？

🔄 暗差原理

拿10个后相等，说明原来第一筐比第二筐多10×2=20个

✅ 解析

和：120，差：10×2 = 20

第一筐：(120+20)÷2 = 70个

第二筐：70-20 = 50个

【范例9：日常消耗问题】

食堂里有94千克面粉，138千克大米，每天用掉面粉和大米各9千克，几天后剩下的大米是面粉的3倍？

【范例9：日常消耗问题】

食堂里有94千克面粉，138千克大米，每天用掉面粉和大米各9千克，几天后剩下的大米是面粉的3倍？

🎯 关键思路

同时消耗相同数量，差值保持不变：138-94=44

✅ 解析

差：138-94 = 44，倍数：3

面粉：44÷(3-1) = 22千克

需要：(94-22)÷9 = 8天

【范例10：调配问题】

甲、乙两个仓库共有大米800袋，如果从甲仓库中取出25袋放到乙仓库中，则甲仓库比乙仓库还多8袋。两个仓库原来各有多少袋大米？

【范例10：调配问题】

甲、乙两个仓库共有大米800袋，如果从甲仓库中取出25袋放到乙仓库中，则甲仓库比乙仓库还多8袋。两个仓库原来各有多少袋大米？

🔄 复杂转化

调配后甲比乙多8袋，原来甲比乙多25×2+8=58袋

✅ 解析

和：800，差：25×2+8 = 58

甲：(800+58)÷2 = 429袋

乙：800-429 = 371袋

【范例11：多倍关系问题】

王老师去慢跑，两天共跑了1000米，已知第二天跑的路程是第一天的2倍多100米，请问第二天比第一天多跑了多少米？

【范例11：多倍关系问题】

王老师去慢跑，两天共跑了1000米，已知第二天跑的路程是第一天的2倍多100米，请问第二天比第一天多跑了多少米？

🎯 化简策略

假设第二天少跑100米，就变成标准和倍问题

✅ 解析

化简后和：1000-100 = 900，倍数：2

第一天：900÷(2+1) = 300米

第二天：300×2+100 = 700米

多跑：700-300 = 400米

【范例12：复合倍数问题】

两组学生参加义务劳动，甲组的人数是乙组的3倍，而乙组的人数是甲组的3倍少40人，问两组学生共多少人？

【范例12：复合倍数问题】

两组学生参加义务劳动，甲组的人数是乙组的3倍，而乙组的人数是甲组的3倍少40人，问两组学生共多少人？

🔄 等量代换

甲=3乙，乙=3甲-40，所以甲=3(3甲-40)=9甲-120

✅ 解析

化简得：甲=9甲-120，所以8甲=120

甲组：120÷8 = 15人

乙组：15÷3 = 5人

共：15+5 = 20人

【范例13：三数和差问题】

大象、老虎、猴子三只动物的年龄中，大象和老虎共90岁，大象和猴子共70岁，老虎和猴子共40岁，请你算一算，三只动物各多少岁？

【范例13：三数和差问题】

大象、老虎、猴子三只动物的年龄中，大象和老虎共90岁，大象和猴子共70岁，老虎和猴子共40岁，请你算一算，三只动物各多少岁？

🎯 巧妙方法

三个和式相加得到2倍的总和，再分别减去各个和式

✅ 解析

总和：(90+70+40)÷2 = 100岁

大象：100-40 = 60岁

老虎：100-70 = 30岁

猴子：100-90 = 10岁

【范例14：多级倍数问题】

三个班去种树，一共种了293棵。已知二班种的数量是一班的2倍多1棵，三班种的数量又是二班的2倍多3棵，问三个班各种了多少棵？

【范例14：多级倍数问题】

三个班去种树，一共种了293棵。已知二班种的数量是一班的2倍多1棵，三班种的数量又是二班的2倍多3棵，问三个班各种了多少棵？

📏 份数分析

一班1份，二班2份+1，三班4份+5，总共7份+6

✅ 解析

纯份数和：293-1-5 = 287棵

一班：287÷7 = 41棵

二班：41×2+1 = 83棵

三班：83×2+3 = 169棵

【范例15：多人份数问题】

甲、乙、丙三个人合做一批零件，甲比乙多做12个，丙做的比甲的2倍少20个，比乙做的多38个。这批零件共有多少个？

【范例15：多人份数问题】

甲、乙、丙三个人合做一批零件，甲比乙多做12个，丙做的比甲的2倍少20个，比乙做的多38个。这批零件共有多少个？

🔄 复杂关系

用乙为基准：甲=乙+12，丙=2甲-20=2乙+4

✅ 解析

乙、丙差：38，份数差：1+4=5，所以乙=(38-4)÷1=34

乙：34个

甲：34+12 = 46个

丙：2×34+4 = 72个

共：34+46+72 = 152个

知识自测

1. 什么是"和差倍问题"？

已知两个数的和、差、倍三个量中的两个，求这两个数分别是多少的问题。

2. 解决和差问题时，求较大数和较小数的公式分别是什么？

较大数=(和+差)÷2，较小数=(和-差)÷2

3. 在和倍问题中，"和÷(倍数+1)"计算出的是哪个数？为什么

计算出的是较小数。因为较小数是1份，较大数是倍数份，总共是"倍数+1"份。

4. 在差倍问题中，"差÷(倍数-1)"计算出的是哪个数？为什么

计算出的是较小数。因为较大数比较小数多"倍数-1"份。

5. 解决和差倍问题，除了公式，还有哪些数学思想或方法？

线段图、等量代换、份数思想、转化法。

6. 【范例8】中，"如果从第一筐中拿10个放到第二筐中，那么两筐的梨子个数相等"这一条件是如何转化的？

转化为"差：10×2=20"，因为第一筐给第二筐10个后相等，说明原来第一筐比第二筐多20个。

7. 【范例7】中，"香蕉比苹果多了4倍"中的"4倍"在计算时为何变成了"5"？

"多了4倍"意味着香蕉是苹果的4+1=5倍。

8. 【范例14】是一个多量（三班）和倍问题，解题时如何设定"份数"来简化计算？

将一班设为"1份"，二班是"2份+1"，三班是"4份+5"，总共"7份+6"。

9. 什么是"暗和"和"暗差"口诀？

暗和：给来给去和不变；暗差：同增同减差不变。

10. 和差倍问题的解题口诀是什么？

有倍先画倍、再找和与差；和、差的时间状态要与倍保持一致。
