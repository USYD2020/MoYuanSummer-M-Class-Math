【范例8：购物问题】

学校第一次买了3个水瓶和20个茶杯，共用去134元；第二次又买了同样的3个水瓶和16个茶杯，共用去118元。问茶杯和水瓶的单价各是多少元？

🌉 消元法

两次购买情况相减，消除水瓶数量

✅ 解析

3瓶+20茶=134，3瓶+16茶=118

两式相减：4茶=16，即1个茶杯=4元

一个水瓶=(134-20×4)÷3=18元

【范例10：书包问题】

用笔袋的价格统一表示

【范例11：饮料购买】

第一天小明买了5瓶可乐和9瓶雪碧用了51块钱，第二天，小明买了2瓶可乐和3瓶雪碧，用了18块钱，那么可乐的单价是多少钱？

🌉 消元法

制造等量关系消除雪碧

✅ 解析

5瓶可乐+9瓶雪碧=51

2瓶可乐+3瓶雪碧=18

将第二个式子乘以3：6瓶可乐+9瓶雪碧=54

两式相减：1瓶可乐=54-51=3元

【范例13：服装价格】

一件上衣和一条裤子共178元，一条裤子和一双鞋共132元，一件上衣和一双鞋共210元。问上衣、裤子、鞋子各多少钱？

🌉 消元法

三个式子相加，求出总价

✅ 解析

上衣+裤子=178，裤子+鞋子=132，上衣+鞋子=210

三式相加：2上衣+2裤子+2鞋子=520

上衣+裤子+鞋子=260

上衣=260-132=128元，裤子=260-210=50元，鞋子=260-178=82元

📝 一页记忆卡（随身背）

一、核心概念

两个完全相等的量可以互相替换，这就叫等量代换。

二、等式三招（口诀：传-加-乘）

  • 1. 传递：a = b，b = c → a = c
  • 2. 加减：两边同加减，等式不变
  • 3. 乘除：两边同乘除（0除外），等式不变

三、多元解题三步曲（建 → 消 → 化）

  1. 1. 建等式
  2. 2. 加减消元（系数变相同，一减就消）
  3. 3. 化单位量（把主要量化成1个、代入其他等式）
